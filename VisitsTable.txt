Visits Table Help Screen:

The Navigation Section:
The Visits Table is intended to show all Visits and their related data. 

The navigation buttons (First, Next, Previous, Last) are to aid with table navigation. 
Each will show the corresponding row in the table

Options and Save Options sections:
Add, edit and delete Patrons using the corresponding buttons
